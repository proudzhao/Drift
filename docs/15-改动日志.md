# Drift 改动日志

本文记录 Drift 在当前已发布版本基础上的简要改动。每次代码、文档、配置或发布相关流程发生修改后，都应追加一条记录。

## 记录规则

- `已发布版本` 填写本次改动基于的最新已发布版本；当前为 `v0.6.2`。
- `日期` 使用本地日期，格式为 `YYYY-MM-DD`。
- `改动描述` 用一句话说明修改内容，优先描述用户可感知影响或维护边界变化。
- 新记录追加到表格顶部，保持倒序。
